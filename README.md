# Diego Claros - QA Automation Portfolio

This repository is a technical portfolio focused on **QA Automation**, built to show practical skills beyond a traditional CV.

## What this portfolio includes

- **CV Website** built with HTML, CSS, and JavaScript
- **UI Automation Framework** using Selenium + Pytest + Page Object Model
- **API Automation Framework** using Python + Requests + Pytest
- **Test Case Samples** written with a QA-friendly structure
- **Starter reporting and CI-ready structure**
