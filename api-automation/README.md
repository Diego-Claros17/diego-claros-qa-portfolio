# API Automation Framework

This project contains a starter API automation framework built with **Python + Requests + Pytest**.

## Features

- Reusable request session
- Simple endpoint abstraction
- Positive and negative tests
- Clean assertions and maintainable structure

## Installation

```bash
pip install -r requirements.txt
```

## Run tests

```bash
pytest -v
```

## Notes

- The example uses the public ReqRes API.
- This project can be extended with schema validation, authentication helpers, and environment config.
