# UI Automation Framework

This project contains a starter UI automation framework built with **Selenium + Pytest + Page Object Model**.

## Features

- Clean project structure
- Reusable page objects
- Base page methods
- Login test example using SauceDemo

## Installation

```bash
pip install -r requirements.txt
```

## Run tests

```bash
pytest -v
```

## Notes

- The example uses Chrome in headless mode.
- You can expand this project with screenshots, Allure, data-driven testing, and CI pipelines.
